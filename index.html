About

I'm a BS/MS Computer Science student at Georgia Tech interested in machine learning, systems, applied AI, and backend engineering.

One idea that fascinates me is how unbelievably dumb computers actually are. At the lowest level they simply flip billions of transistors on and off. Yet somehow we've learned how to turn that into systems that recognize images, understand language, and learn incredibly complex patterns.

Studying low-level systems has given me a deeper appreciation for the software we build. Through internships and research, I've worked on applied AI, machine learning, and backend systems, and I'm excited to continue exploring the intersection between intelligent software and the systems that power it.

Favorite Ideas in Computing
Virtual Memory systems
The illusion that programs have infinite memory through paging and address translation.
Hash Maps data structures
O(1) average lookup changes everything about how you think about problems.
Cache Hierarchy systems
Why algorithms that touch memory sequentially can be 10–100× faster. Don't use linked lists.
Gradient Descent ml
Another goat. The way it handles insane optimization landscapes still feels like magic.
Attention ml
Everything interacts with everything else through similarity. Blew my mind the first time I learned it.
Embeddings ml
Abstract concepts become points in space — and suddenly you can do math on meaning.
Backtracking algorithms
The way recursive DFS explores entire decision spaces systematically is genuinely beautiful.
False Sharing concurrency
Threads interfering just by touching nearby memory. A performance bug invisible at the code level.
